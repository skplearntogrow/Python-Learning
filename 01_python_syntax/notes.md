**Learnt GIT Commands:**
git --version -> check git version
git init -> Initialize git
git remote add origin Github HTTPS repo url -> set connection local to remote repo
git branch -M main -> renames your current branch to specific branch name "main". "M" stands for rename or move forcefully and does not ask confirmation. "-m" Git will throw an error if you accidentally try to overwrite an existing branch. Here "-" means Short-form Indicator.
git branch -> see which branch you are On.
git add . -> stages all changes
git commit -m "commit message of changes performed" -> save changes locally
git push -u origin main -> uploads commits to Github main branch. Here "-u" stands for set-upstream
git push -> uploads local commits to default branch in github
git config --global user.name "My full name" -> user name set.Git Credential Manager usually stores your credentials securely.
git config --global user.email "My email address that is used for Github account" -> email id is set.Git Credential Manager usually stores your credentials securely.
git config --global --list -> list all the global user credential details
git status -> see current status

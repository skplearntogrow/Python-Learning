**Topics Covered:**
1. In Python Comments are used for In-Code documentation
2. Comments can be used to explain a statement or provide pre-requisite for next statement or program objective.
3. Comments can also be used to not execute a line of code during Testing.
4. Comments are of 4 Types:
    a. Single line comment
    b. In-line comment
    c. Multiline comments
    d. Multiline String comments
5. As long as Multiline string is not assigned to Variable, python will consider it as a comment.

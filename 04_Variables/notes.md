**Topics Covered:**
1. Variables are containers to store data values
2. Python does not have a command to declare a variable.
3. Variables are created when a value is assined for first time.
4. Variables do not need to be declared with particular type. The type can be changed after they are set. As python execute immediate after each line, python takes new value as latest for the same variable already value was set
5. To specify data type of a variable is done by Casting
6. If we want to get the data type of a variable, use type()
7. String variables can be declared using single or double quotes.
8. Global Variables and Local Variables declaration and usuage.
9. Function names are Case Sensitive same as Variable names
10. Changing Global variable value within a function scope.
11. def function scope creation and execution.
12. Create a variable outside function and use it inside function.
13. Create a variable inside function with same name as global variable
14. If we use global keyword for variable within a function,it will be a global scope.
15. To change a value of a global variable inside a function, use global keyword.
16. print() function is used to output variables
17. In print() function we can output multiple variables using a comma.
18. You can also use + Operator to output multiple variables.However + Operator is used only when all the variables are of same data type.
19. For numbers, + Operator in print() function acts as Mathematical addition.
20. if you want to output both integer value with string value in print() function together then use , operator.
21. The best way to output multiple variables in print() function is to use , Operator.
22. Multi Word Variable names: Variable names are Case-Sensitive. Follow Camel case or Pascal case or snake case.
23. Variable names should not start with number
24. Variable names can start with _ character
25. Variable names are alpha numeric,string or _ underscore characters.
26. Variable names cannot be a python keyword.

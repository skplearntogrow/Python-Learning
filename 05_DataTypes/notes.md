**Data Types Topics Covered:**
1. Variables can store data of different data types
2. Set data type when declaring a variable. Ex: x=int(5)
3. To know the type of datatype of Variable use type().Ex: print(type(x)). This is called as Casting.
4. In Python, below are the different Built-in data types:

    1. Text type: str
    2. Numberic type: int,float,complex
    3. Sequence type: list, tuple, range
    4. Mapping type: dict
    5. Set type: set, frozenset()
    6. Boolean type: bool
    7. Binary type: bytes, bytearray, memoryview
    8. None type: NoneType

**Topics Covered:**

1. There are 3 numeric types in Python - int,float and complex
2. Variables of numeric type are created when you assign value to them.
3. To Verify type of any object use type() function.
4. Int or Integer: Is a whole number, positive or negative without decimal and with unlimited length
5. float or floating point number: This is a decimal number, positive or negative.
6. complex numbers are written with "j" as Imaginary part.
7. Type Conversion: Python allows us to convert from one type to another with int(),float() and complex() functions.
    Note: We cannot convert complex number to another numeric type
8. Python does not hav random function to create random number, but python has a built-in module called random that can be used to make Random numbers.
